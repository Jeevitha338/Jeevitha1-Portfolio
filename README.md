# Professional Personal Portfolio

A clean, modern, and fully responsive personal portfolio website built with vanilla HTML5, CSS3, and JavaScript. No frameworks or libraries required.

## 🚀 Features
- **Modern Design:** Professional look with a clean layout and good spacing.
- **Fully Responsive:** Optimized for mobile, tablet, and desktop.
- **Smooth Navigation:** Sticky navigation bar with smooth scrolling between sections.
- **Interactive Elements:** Hover effects, subtle animations, and a scroll-to-top button.
- **Skills Visualization:** Progress bars for technical skills.
- **Experience Timeline:** Professional career history in a timeline layout.
- **Contact Form:** Functional-ready form with client-side validation.
- **SEO Ready:** Optimized with semantic HTML and meta tags.

## 🛠️ Local Development

### Option 1: Using Visual Studio Code (Recommended)
1. Open the `portfolio` folder in VS Code.
2. Install the **Live Server** extension (by Ritwick Dey).
3. Right-click on `index.html` and select **"Open with Live Server"**.
4. The website will automatically open in your default browser.

### Option 2: Direct Opening
1. Navigate to the `portfolio` folder.
2. Simply double-click on `index.html` to open it in any web browser.

## 🌐 Netlify Deployment

### Option 1: Drag-and-Drop (Easiest)
1. Create a free account at [Netlify](https://www.netlify.com/).
2. Log in and go to the **Sites** tab.
3. Drag the entire `portfolio` folder into the deployment box.
4. Your site will be live in seconds!

### Option 2: GitHub Integration (Best for Updates)
1. Push your portfolio code to a GitHub repository.
2. In Netlify, click **"Add new site"** > **"Import an existing project"**.
3. Connect your GitHub account and select the repository.
4. Click **"Deploy site"**. Netlify will automatically redeploy whenever you push changes to GitHub.

## ✏️ Customization
To personalize this portfolio, open the files and look for comments:
- **`index.html`**: Update your name, title, bio, education, skills, experience, and contact details.
- **`css/style.css`**: Modify the `:root` variables at the top to change the color scheme.
- **`assets/`**: Replace the placeholder images with your actual photos.
- **`resume/`**: Replace `resume.pdf` with your own professional resume.
